So far we have November 14th/15th. Would love for BB to be able to come, so let's say first choice 14th, second choice 15th.

We haven't ever done anything at Lake Houston park, so that might be a possibility. But I can't remember if there's a good gathering area.

If someone who is a more regular poster on the Texas list wants to write a post steering those interested in helping plan to the discussion here that would be swell.
